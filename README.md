# Kirnagram Frontend

Frontend application for Kirnagram built with React, Vite, TypeScript, shadcn-ui, and Tailwind CSS.

## Setup

```sh
cd kirnagram-frontend
npm install
npm run dev
```

## Scripts

- `npm run dev`: Start local development server
- `npm run build`: Production build
- `npm run preview`: Preview production build
- `npm run lint`: Run eslint
- `npm run test`: Run tests once
- `npm run test:watch`: Run tests in watch mode

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
